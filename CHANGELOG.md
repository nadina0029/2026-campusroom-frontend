# Changelog

Semua perubahan penting pada proyek ini akan didokumentasikan dalam file ini.

Format berdasarkan [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
dan proyek ini mematuhi [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-17
### Added
- Inisialisasi proyek menggunakan React + TypeScript + Vite.
- Integrasi **SweetAlert2** untuk notifikasi interaktif.
- Implementasi **Axios** untuk komunikasi dengan Backend API.
- Halaman **Login & Register** dengan validasi.
- Halaman **Dashboard** dengan fitur:
  - Filter pencarian ruangan.
  - Custom Luxury Dropdown.
  - Tabel data ruangan (Glassmorphism style).
- Halaman **Riwayat Peminjaman** (Booking History).
- Fitur **Admin Console**: CRUD Ruangan (Tambah, Edit, Hapus, Kunci Status).
- Fitur **Booking Modal**: Form pengajuan peminjaman untuk mahasiswa.

### Changed
- Perombakan total UI/UX menjadi tema **"Midnight Elite"** (Dark Mode + Aurora Glow).
- Penggunaan Bahasa Indonesia pada seluruh antarmuka aplikasi.
- Perbaikan kontras warna teks untuk aksesibilitas yang lebih baik.
