# 2026-campusroom-frontend 🏫✨

**Frontend Application** untuk Sistem Manajemen Peminjaman Ruangan Kampus. 

## 🌟 Fitur Utama
* **Real-time Dashboard**: Melihat status ketersediaan ruangan secara langsung.
* **Booking Portal**: Formulir pengajuan peminjaman ruangan yang mudah bagi mahasiswa.
* **Admin Console**: Fitur lengkap untuk Admin (Tambah/Edit/Hapus Ruangan, Kunci Ruangan, Approval Peminjaman).
* **Interactive Notifications**: Umpan balik visual menggunakan SweetAlert2.

## 🛠️ Teknologi yang Digunakan
* **Framework**: React.js (Vite)
* **Language**: TypeScript
* **HTTP Client**: Axios
* **Styling**: CSS-in-JS (Inline Styles & CSS Modules)
* **Alerts**: SweetAlert2

## 📋 Prasyarat
Pastikan Anda telah menginstal:
* [Node.js](https://nodejs.org/)
* NPM atau Yarn
